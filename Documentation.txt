#YouTube downloader



Version 1.1.0 (actual)
In this version user only needs to execute the main.py. He copy/paste the link of the video
he/she wants to download in the file links.txt. Users only input if they want to download
audio or video of the links in the links.txt file.


Version 1.0.0
The main branch of the project allows the user modify the variable url, then
user choose audio or video download. Finally, he/she selects quality of the video. 
Also user input the if he wants audio or video and the quality in both of them
